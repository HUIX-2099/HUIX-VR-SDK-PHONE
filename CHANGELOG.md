# Changelog

All notable changes to HUIX Phone VR SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-07

### Added
- Initial release of HUIX Phone VR SDK
- **Core Systems**
  - HUIXVRManager - Central VR system manager
  - HUIXVRCamera - Stereoscopic camera rendering
  - HUIXHeadTracker - Gyroscope-based head tracking
  - HUIXInputManager - Gaze and trigger input handling
  
- **Visual Components**
  - HUIXReticle - Customizable gaze cursor
  - Lens distortion correction shader
  - Chromatic aberration correction
  - Stereo composite shader
  
- **UI System**
  - HUIXVRButton - Gaze-interactive button
  - HUIXVRSlider - Gaze-interactive slider
  - HUIXVRCanvas - World-space VR canvas
  
- **Utilities**
  - HUIXVRRig - Complete VR rig setup
  - HUIXTeleporter - Gaze-based teleportation
  - HeadsetProfile - Configurable headset profiles
  
- **Editor Tools**
  - Setup Wizard for easy scene configuration
  - Custom inspectors for all components
  - Quick Setup menu option
  - Headset profile editor with preview
  
- **Samples**
  - Demo scene showcasing all features
  - Interactive object examples

### Headset Profiles
